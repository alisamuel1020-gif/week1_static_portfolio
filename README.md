## ğŸ’» Local Setup

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari)
- Text editor (VS Code, Sublime Text, etc.)
- Git (optional, for cloning)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/alisamuel1020-gif/week1_static_portfolio.git
```

2. Navigate to the project directory:
```bash
cd week1_static_portfolio
```

3. Open `index.html` in your web browser:
```bash
# On Linux/Ubuntu:
xdg-open index.html

# On Mac:
open index.html

# On Windows:
start index.html
```

No build process or dependencies required - just open the HTML files!
